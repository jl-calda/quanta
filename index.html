For Claude Code

All files sit at the project root; shared helpers are gallery-thumbs.jsx (nav rail, icons, math thumbnails) and the per-screen *-app.jsx files. The editor splits into editor-icons / editor-canvas / editor-app.jsx. Density is driven by density.css + density.js (Comfortable / Compact, default Compact). The design system bundle and tokens live under _ds/ — load order is React → Babel → bundle, then compose with window.QuantaDesignSystem_019e2c.
